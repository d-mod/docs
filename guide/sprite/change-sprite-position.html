    # 修改贴图坐标

    选择需要修改的贴图坐标,支持多选

    点击右侧贴图列表右键菜单中的修改贴图坐标

    TIP

    相对坐标是指在原坐标的基础上加该坐标值

    勾选所有贴图可以修改所有贴图

    # 便利性操作

    使用Ctrl+B快捷键可以将贴图的坐标修改为当前在画布中的位置
