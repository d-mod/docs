    # 修改贴图大小

    选择需要修改的贴图,支持多选

    点击右侧贴图列表右键菜单中的修改贴图大小

    选择修改方式

    # 自定义贴图大小

    TIP

    自定义贴图大小是将贴图大小改为自定义的宽高，并将原贴图在新贴图中的位置设为贴图的坐标

    # 按比例缩放

    TIP

    按比例缩放会将贴图按自定义的比例缩放,单位为%

    # 裁剪透明画布

    TIP

    裁剪透明画布是自定义贴图大小的逆向操作,会将贴图边缘的透明部分裁剪掉
